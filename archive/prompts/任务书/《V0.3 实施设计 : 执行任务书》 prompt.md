	•	summary

	•	resolvedProfile
	•	issues
	•	matrices
	•	artifactIndex
	•	reportMarkdown
	•	unresolvedFacts

	•	visibility 是 top-level canonical visibility contract
	•	manualReviewNeeded 是 canonical 人工复核布尔语义
	•	issue 明确输出 issueKind / applicabilityState / evidenceMissing / manualReviewReason

当前仍明确：
	•	PDF 为 pdf_text_only + parserLimited=True
